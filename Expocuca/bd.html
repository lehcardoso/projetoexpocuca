Banco de Dados

A matéria de Banco de Dados ensina como gerenciar dados de forma eficiente. Os alunos aprendem a modelar dados, utilizando conceitos como entidades, atributos e relacionamentos. Também exploramos SQL utilizando o MySQL. Além disso, fomos introduzidos a diagramas, como o Diagrama Entidade-Relacionamento, que ajuda a visualizar a estrutura e a organização dos bancos de dados.



O que foi aprendido
