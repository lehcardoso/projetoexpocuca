Análise e Projeto de Sistemas

A matéria de Programação Web aborda a criação e manutenção de sites e aplicações web, começando com os fundamentos de HTML e CSS para construir e estilizar páginas. Os alunos aprendem a estruturar conteúdo com HTML e aplicar estilos e layouts com CSS, incluindo técnicas de design responsivo para otimizar a visualização em diferentes dispositivos.
